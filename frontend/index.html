    Under The Hood

    How it works

    Data

    Transaction features

    Amount, distance from home, distance from the last transaction, purchase ratio, retailer history, and how the card was used — chip, PIN, or online.

    Model

    Random Forest classifier

    scikit-learn, 300 trees, class-weighted to handle the ~3% fraud imbalance. Trained and served with a Flask REST API.

    Interface

    This dashboard

    Calls the live API at localhost:5000 when it's running; otherwise falls back to an equivalent scoring model running in your browser, so the demo always works.
